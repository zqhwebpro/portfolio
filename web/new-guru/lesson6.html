Microformats

In a nutshell, microformats are encoded HTML markup used to organize important data on a website. For instance, microformats can be used as business cards, event data, and other types of important data to be extracted from the HTML code and plugged into other types of software or apps. So that business card can be viewable by search engines, the event data can be sync'd to a calendar, and other important data even though not completely important is associated with the webpage.

vCard


vCard is a format best used for transferring contact information. In the industry, web developers use vCard to exchange electronic business cards and email signatures.

Example:

BEGIN:VCARD
VERSION:3.0
N:Man;Example
FN:Example Man
ORG:Example Design Agency
TITLE:Cool Dude
PHOTO;VALUE=URL;TYPE=JPG:http://www.example.com/employee_photos/exampleman.jpg
TEL;TYPE=WORK,VOICE:(717) 818-4455
ADR;TYPE=WORK:;;654 Example Rd;Example;PA;17406;United States of America
LABEL;TYPE=WORK:654 Example Rd\nExample, PA 17406\nUnited States of America
EMAIL;TYPE=PREF,INTERNET:exampleman@example.com
END:VCARD

After we save that file as a .vcf or a .vcard file then it will be set up to transfer key contact details from your webpage to another source.

hcard


hcards embed data in HTML to easily add contacts, save event dates, and other more advanced contact details like that.

Example:

<div class="vcard">
    <a class="fn org url" href="www.example.org" >Example </a>
        <div class="adr">
            <span class="type">Work </span>
            <abbr class="region" title="New York">NY</abbr>
        </div>
        <div class="tel">
            <span class="type">Work </span> 1-717-818-4455
        </div>
</div>

By using HTML embedded data I have created an hcard explaining some generic work details to inform search engines, users, or other third-party attributes. I was able to do this by calling a vcard class and then using vcard properties for the embedded information from Microformats.org, a great link to bookmark!

ical


ical transfers meeting request, event times, or things of that nature. It helps plan dates and times and is supported by many calendar setups on the World Wide Web.

Example:

BEGIN:VCALENDAR
VERSION:2.0
PRODID:-//hacksw/handcal//NONSGML v1.0//EN
BEGIN:VEVENT
UID:uid1@example.com
DTSTAMP:20150304T3000Z
ORGANIZER;CN=Example Man:MAILTO:exampleman@example.com
DTSTART: 20150304T12000Z
DTEND:20150304T16000Z
SUMMARY:Tom Dick Harry's Birthday Celebration
END:VEVENT
END:VCALENDAR

Calling the vcalendar function allows me to set a date time for a birthday celebration to share with the intended attendees so that everyone can make plans accordingly.

vcalendar


vcalendar is like ical however, it utilizes HTML to create transferable calendar data. It is like the hcard to the vcard. There is more customization available in concerning embedded HTML information but it serves its own separate purpose for that very same reason.

Example:

<p class="vevent">
    The <span class="summary">Tom Dick Harry's Birthday Celebration</span>
    on the 4th of March 2015 begining
    <abbr class="dtstart" title="2015-03-15T14:00:00+012:00">12</abbr>am and ending at
    <abbr class="dtend" title="2015-03-15T14:00:00+016:00">4</abbr>pm. Located at
    <span class="location">Chucky Cheese</span>!
</p>

That is the same series of information from the ical example reformatted for specific vcalendar requirements. More properties for the event vcalendar class can be located at A Wikipedia list of calendar properties.

Make your own vCard business card! Step-by-step Tutorial.


Step 1 - File New!

Begin by creating a new file in the .html software of your choice.

Step 2 - Code your Vcard!

Begin by calling the Vcard element by typing BEGIN: VCARD and hit return. Next goto this vcard wikipedia website and add the properties to your code that you want to add to this digital business card. I'll share my example:

BEGIN:VCARD
VERSION:3.0
N:Man;Example
FN:Example Man
ORG:Example Design Agency
TITLE:Cool Dude
PHOTO;VALUE=URL;TYPE=JPG:http://www.example.com/employee_photos/exampleman.jpg
TEL;TYPE=WORK,VOICE:(717) 818-4455
ADR;TYPE=WORK:;;654 Example Rd;Example;PA;17406;United States of America
LABEL;TYPE=WORK:654 Example Rd\nExample, PA 17406\nUnited States of America
EMAIL;TYPE=PREF,INTERNET:exampleman@example.com
END:VCARD

That which I had wished to include has been called out in that string of code. Everything I include will show up on my vcard.

Step 3 - File Save

Save your file AS you pick.VCF to let browsers know it is a vcf file.

Step 4 - Upload Vcard

Upload the vcard to a Web server so that it can be viewable by all that you share it to on the internet.
