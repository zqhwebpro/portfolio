CSS3 Advanced Selectors

Objective:

In this lesson I will teach you a few key CSS3 selectors and how to use them to make your websites look amazing and be more functionally structured.

Sub-objectives:

Understand the CSS3 Attribute Selectors: a, ~, |, $

Learn the CSS3 Psuedo-Selectors: :first-child, :nth-child, :before, & :after

Prerequisites:

For this lesson you will learn the best if you have prior advanced CSS3 knowledge and HTML5 will come in handy in putting the CSS into effects.


*

* selects all elements that it targets. Let's take a deeper look:

*p //selects all of the p elements

Try * for yourself!

~

~ will initiate a search call-out. Let's take a deeper look:

[title~=cat] {
    background-color: red;
}

Try ~ for yourself!

|

| will find anything that has the class input called upon. Let's take a deeper look:

[lang|=en] //finds all of the elements assigned to call out the language of English

Try * for yourself!

$

$ will find matches. Let's take a deeper look:

a[href$=".jpg"] //finds all the .jpg's in your document

Try | for yourself!

:first-child & second-child

first child changes the FIRST child in the callout function. That is, without any exception, a call-out that will initiate the FIRST child of ANY parent! Let's take a deeper look:

p:first-child {
    background-color:yellow; //highlights in yellow starting from the first child of p
}

p:second-child {
    background-color:yellow; //highlights in yellow starting from the second child of p
}

Try first-child for yourself!

nth-child

nth works like the first child except you have to input the nth of the parent. The standardized equation for nth goes like this (xn+y) x=the multiple property of n + y which = a plus factor. Let's take a deeper look:

p:nth-child(4) {
    background-color:yellow; //highlights from every four children
}

Try nth-child for yourself!

::before ::after

before and after insert content before or after an element. Let's take a deeper look:

a:before {
    width:100px;
    height:100px;
    background-color:blue;
}

a:after {
    width:100px;
    height:100px;
    background-color:blue;
}

Try ::before and ::after for yourself!
