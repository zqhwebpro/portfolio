Animation Examples

Follow along as I demonstrate alternatives to javascript using only CSS. Javascript creates a lot of movement on a webpage but there are ways to code it completely in CSS. This will cause less initial scripts to be necessary to run and quicker load times.

Wooow! Fade block!


Test this out, wooow!, It fades.


To make a block fade with have to call out both an id and a class on a div. The opacity property will alter the fade when set at 0 to be fully transparent. When we hover over it, it will disappear under the cursor. It's a few CSS callouts to set something up on your website like this. It goes likes this:

/*--IN CSS--*/

#phase {
    opacity: 1;
    transition: opacity 1s linear 0s;
}

.block {
    border: 6px solid #ccc;
    padding: 11px;
}

#phase:hover {
    opacity: 0;
}

That will basically set up the div to go to 0 in opacity on hover. You just set up the HTML to callout both functions and it will phase out.

Second effect: Pulsation Hover


Pulsating block div.


The pulsating div block works like magic to draw attention to important information. By calling on a -webkit-keyframe property I was able to animate the pulsation effect. It goes likes this:

/*--IN CSS--*/

#pulsate {
    height: 110px;
    width: 110px;
}

#pulsate:hover {
    -webkit-animation-name: pulsate;
    -webkit-animation-duration: 10s;
    -webkit-animation-timing-function: ease-in-out;

    -moz-animation-name: pulsate;
    -moz-animation-duration: 10s;
    -moz-animation-timing-function: ease-in-out;

    -o-animation-name: pulsate;
    -o-animation-duration: 10s;
    -o-animation-timing-function: ease-in-out;
}

@-webkit-keyframes pulsate {
    0% { width:150px;}
    5% { width:160px; left:-1px;}
    10% { width:150px; left:0px;}
    15% { width:160px; left:1px;}
    20% { width:150px; left:0px;}
    40% { width:160px;}
    45% { width:150px; left:-1px;}
    50% { width:160px; left:0px;}
    55% { width:150px; left:1px;}
    60% { width:160px; left:0px;}
    80% { width:150px;}
    100% { width:160px;}
}

Now, most, if not all, browsers will active a pulsate function.

Third effect: Nudge


Please don't mouse over me and nudge me!


Nudge uses a transition property and duration, so it has to hover that initiates it's change. It goes likes this:

/*--IN CSS--*/

#nudge{
        -webkit-transition-property:color,background-color,padding-left;
        -webkit-transition-duration:230ms,230ms,2300ms;

        -moz-transition-property:color,background-color,padding-left;
        -moz-transition-duration:230ms,230ms,2300ms;

        -o-transition-property:color,background-color,padding-left;
        -o-transition-duration:230ms,230ms,2300ms;
}

#nudge:hover{background-color:#333;color:white;padding-left:230px}

Things have been nudged.

I expand if touched.

Effect Five: Bounce


All Bouncyfun Palaces, 50%off!


Margin was used for the bounce keyframe query causing the information below it to change along with it in this case. And it goes something like this:

#bounce{

With a timed duration, iteration, and direction the bounce hover now keyframes to our specifics.

#6: The Spin Block


On sale today!


The spin block creates a rotation style of animation transforming onto your website. It's a great way to accentuate important features. It goes like this:

/*--IN CSS--*/

#spinblock {
    -webkit-transition: -webkit-transform 7s ease-in;
}

#spinblock:hover {
    -webkit-transform:rotate(360deg)
}

-webkit- Browsers will compute the transform and transition to make it ease-in to a full circle rotation.

D Open Theory

When travelling to D Open on chord progression you have to find your specific style, strumming pattern, and creative outlook on the entire thing.



In order to create an accordion effect set the id. Then use CSS to change it to your will. Mine goes like this:

/*--THE CSS--*/

#accordion div{
    height:0;overflow:hidden;
    -webkit-transition:height 230ms ease;
}

#accordion div:target {
    height:140px;
}

.accordion a {
    display:block;
    padding:5px 10px;
    background-color:#333;
    color:#eee;
    text-decoration:none;
    }

.accordion a:hover {
    background-color:#999;
}

.accordion div {
    background-color:#ccc;
    color:#222;
}

.accordion div p {
    padding:20px;
}

Setting the id and class give you a lot of control over your menu structures, animation, and CSS3 in general.
