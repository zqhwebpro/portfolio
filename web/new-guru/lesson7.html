SEO Basics

Objective:

In this lesson you will learn what search engine optimization (SEO) is and how to use SEO to improve your websites search ranking.

Sub-objectives:

Understand some of SEO's best pracrices

Learn how to use meta tags to describe your website

Learn the use of robot.txt

Learn the use of "nofollow"

Know how to configure SEO approved mobile sites

Know how to properly promote a website

Learn the use of webmasters tools for proper SEO implemation

Prerequisites:

It is required for this lesson that you have a decent understanding of HTML.


What exactly is SEO?

Search engine optimization is your way of making sure that if a user is searching the internet for content that your website will rank higher on the list than the others. Web developers use SEO to build a websites reputation and validity. By taking the time to properly describe, safeguard, and promote your website it will rank to the top in no time!

The following are the methods to practice to acheive the best SEO results:


Website Titles:

Every website should have a title. The title itself is what will be displayed as the entire website title link when a user finds it within a search engine. For example, a pet store by the name of Zoo Palace that sells only exotic pets is starting a website. In the title it is important we convey to our users the exact definition of what this webpage is promoting. So the best title for this website would be: Zoo Palace - Selling Exotic Pets and Exotic Pet Accessories. Each page in the website should have a unique title name for example, referring to the pet store, a Chinchilla shopping page can be titled Zoo Palace - Selling Exotic Pets and Exotic Pet Accessories - Chinchillas. It's best to be as brief as possible yet still give an accurate description of the website.

meta tags:

meta tags can be added to the websites head section to describe the website in more detail. The tag, if done precisely enough, might actually show under the website link in the search engine as the description of the websites content. A website meta tag for the Zoo Palace example would look something like this.

<head>
<title>Zoo Palace - Selling Exotic Pets and Exotic Pet Accessories</title>
<meta name="description=" content="Zoo Palace is Lancaster County's only pet shop for exotic types of pets and exotic pet necessities.
We also offer training courses and exotic pet grooming services.">
</head>

URLs and Navigation:

When web developers create their websites they use memorable URL names and file structure to make content easier to access for users. By maintaining a simple directory structure and using descriptive enough URL names users will be more likely to remember and search for information that they access on the website. It is very important that your entire website be easier to navigate through or a frustrated user my give up on your content. For example, say a user has accessed a news article on your website that looks like this:

www.zoopalace.com/news/april2015/exotic-pets-help-releive-stress.html

Say that same user has become interested in the other news articles of april2015, by removing the exotic-pets-help-releive-stress.html portion of the URL they should be sent back in the directory to the content that holds all of the April 2015 news articles. With simple and precise directory structure, searching through your website should be a breeze. Other important factors dealing with URLs and navigation include the practice of using an XML sitemap file and having a unique 404 page. These practices ensure the best user experience and will bump your page up in the search results so it is important that you follow these procedures.

Writing for the Web

Next, I'll briefly discuss the important of quality content. When writing content for a website it is best to be clear and concise. Users confused by terms and phrasing will leave the website; damaging the websites reputation. Be exact about your websites content. Also, make proper use of the heading tags. By default a heading goes from the largest size of 1 to the smallest size of 6 and should define the information architecture of the websites accordingly. Heading tags, the websites content, the keywords, and proper image alt text come into play by accurately describing the websites content and help search engines to find quality websites. Let's take another look at Zoo Palace

<h1>Zoo Palace!</h1>
<h2>Zoo Palace - In The News!</h2>
<h3>The Top 10 Reasons Monkeys Make Amazing Pets</h3>
<h4>Written by Kat Daniels - March 3rd, 2010</h4>
<p>Buckle your seat bet monkey lovers, have I got some new information for you.,,</p>

robots.txt

A robots.txt file is the acting agent that will allow or deny a search engine to crawl for information in certain parts of your website. More on how to implement the robots.txt file will come in this lessons tutorial. More information can be found for robots.txt by clicking here: robot.txt information. There are a few commands added to the file but follow the below example and make your own!

These lines of code will block google from crawling this paticular image:

User-agent: Googlebot-Image

Disallow: /images/zoojourney_commmunityphoto.jpg

Save the txt file as robots.txt and place it in the main directory of your website and you're done!

rel="nofollow"

By using the rel="nofollow" attribute you prevent search engines from using the nofollow link data attached as a source of validity. Meaning, if a spammer post on your website in a comment box for a link to their website, a search engine will see the nofollow attribute and know not to associate the link with your websites content. The link would look something like this

<a href="guestcommentbook.html" rel="nofollow">Guest Comment Books</a>

or you can make the link a page level function so that NO links will be traced

<meta name="robots" content="nofollow" />

Mobile SEO Configuration

Search engines crawl for desktop and mobile sites different therefor it is important you make sure that both of these sites can be properly recognized. To do so, make sure you redirect users to a mobile site if they're on a mobile device. Content can be changed to fit the needs of both users, since different information can be more applicable to the mobile community as opposed to the desktop community.

Website Promotion

Promotion is also important for SEO. By using Social Media and getting in touch with communities of people in need of your websites services you advertise. Always keep that in mind! A good reputation isn't always easy to come by in the convoluted information highway that is the internet so create good relations and maintain a friendly attitude.

Webmaster Tools

Webmaster tools provide an analysis of your entire websites infrastructure. These type of tools can tell you how search engines interact with your website and offer feedback on what areas of the website are in need of improvement for the best possible SEO outcome. If you own a website Click here for Google's Webmaster Tools to get a full analysis on how your SEO stacks up!
