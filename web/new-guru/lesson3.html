Media Queries

Objective:

In this tutorial you will learn what media queries are, their benifit to the web industry, and how to effectively use them.

What are media queries and why do web designers use them?

Media queries are a CSS command that acts exactly as it sounds. We are basically starting a media response command. CSS with initiate the command in a user’s automatically depending on different variables which will be discussed later in this lesson. Web designers needed to adapt to the change in device resolutions. As new technologies immerged a growing market demand that the online presence for these devices was satisfied appropriately. Coders added the media query command to CSS to be a way to automatically adjust window sizes accordingly.

@media not (property: value)

the not operator will do anything except the lines of code following it.

@media only (property: value)

the only operator will only do the lines of code following it and nothing else.

@media and (property: value)

the and operator will do both the line of code proceeding it and following it.

@media or (property: value)

the or will do either what is proceeding it or following it depending on the conditions.

@media (min-width: value)

min-width refers to the minimum width of browser window size before the media query executes.

@media (max-width: value)

max-width refers to the maximum width of browser window size before the media query executes.

@media (min-device-width: value)

min-device-width refers to the minimum width of device size before the media query executes.

@media (max-device-width: value)

max-device-width refers to the maximum width of device size before the media query executes.

@media (min-height: value)

min-height refers to the minumum height of browser window size before the media query executes.

@media (max-height: value)

max-height refers to the maximum height of browser window size before the media query executes.

@media (min-device-height: value)

min-device-height refers to the minimum height of device size before the media query executes.

@media (max-device-height: value)

max-device-height refers to the maximum width of device size before the media query executes.

Media query in action:

So lets say we want a computer screen to adjust our web content properties when the Browsers window size reaches a specific variable. That specific line of code would like like so:


@media only screen and (min-width:800px) {
    #media_box{
        background-color:white
    }

Test it, bring the right side of your browser window closer to the left side until the bar turns white.



So here is how that works. In CSS we call out a media query with @media.

Next we add or logical operators so next is only screen. That means "screen" is the only thing effected. Then we call out and. This signifies what only the screen will do. Next we add media feature with a property and a value. So when the screen reaches a minimum of 800px. The next CSS property will continue. Add a CSS command for the parameters to change to and that's it! Next, apply what you've learned with the following tutorial.
