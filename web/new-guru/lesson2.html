icomoon.io and fontawesome.io

Objective:

By the end of the lesson you should have a clear idea on how to use the websites and the benifit to using these website to help you improve yours.

Prerequisites:

You must be experienced with HTML and CSS

Sub-objective:

Why choose font packs instead of graphics? By using font packages like icomoon and font awesome you'll receive a number of benefits. All of the icons are vector based so they are fully sizeable. They are much more crisper then compressed graphic files. They also load quicker, and are fully customizable through CSS.

Setting up icomoon.io

Icomoon always you to sign up for their icomoon.io app service and individually pick from their series of free icons. To add an icon from this website to yours follow these steps:

1. Go to https://icomoon.io/app/#/select and click a few icons to add to your project.

2. When you have what you would like to add, click the generate font icon. Next download the icon pack (the same button as before).

3. Drag the fonts folder into your websites file structure - in to your library folder.

4. Drag the .css style file to your desktop, rename it icomoon and then drag it into your css folder.

5. Next add HTML line of code given to you from icomoon.io by clicking the <> Get Code button under the icon you picked

Now you're able to add an icon available on the free library or even important your own graphics to change into font files. Again, to touch over the benefits of this it will reduce file sizes, give you ultimate control of sizing, and be a very crispy clean graphic to add to your website. Also, both icomoon and font awesome give you the option to alter the CSS code potentially adding even more to your web code. You can see for yourself by inspecting the icon element and altering the code data!

Setting up fontawesome.io

Icomoon always you to sign up for their icomoon.io app service and individually pick from their series of free icons. To add an icon from this website to yours follow these steps:

1. Go to http://fortawesome.github.io/Font-Awesome/ and click the big download button.

2. Extract the contents from their font folder to your font folder, and there css to your css folder.

3. Open the font-awesome.css file and change your @font-face path to the correct pathway for your site structure

4. Next add your icon by referencing this line of code <i class="fa fa-home"></i>

Both font awesome and icomoon are great tools to add to your hyperbole wed designer tool belt. Both are simple to use and will help to maintain a sleek and efficient web page.
