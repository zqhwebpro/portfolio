Lesson 7 - CSS Properties

Objective: The basics are done, the box model has been learned, so our next step is now, ta da da da... CSS Properties. We won't learn the complete list of all of the CSS properties available but we will learn the most used properties.

CSS Resets:

Why resets?

Brosers do not have consistent default properties therefor a CSS Reset must be used in order to fix cross-browser look inconsistencies. A CSS Reset is used to give everything a default look and value. It make be tweaked to fit your needs but it's best to have one in place. You can view a couple default CSS Reset's below.

http://meyerweb.com/eric/tools/css/reset/

http://yui.yahooapis.com/3.2.0/build/cssreset/reset.css



Text Properties

font-family

With font-family you can change the font completely. For example, you can switch the default to Arial or make certain strings of code Veranda.

font-size

In order to set the size of the font, use the font-size tag.

font-weight

Fonts come in multiple weights meaning with a font weight you can set a paticular font to be thinner or bolder and wider with the font-weight tag.

font-style

Some fonts also come with multiple styles meaning certain fonts you can change into a more decorated version of the font or quite possibly a style that offers more kerning. It depends on the font you chose which possibly styles are attached..

color

To change the default color use the color property. You can set colors by using generic terms like #white and #blue. Or you can set the hex web color like this for white: #FFFFFF and blue like this: #0000FF.

text-decoration

There are a few text decorations. For instance, a link by default as a text-decoration of underline. Other text decorations are things like line-through, overline, blink, and inherit. (inherit takes the value of it's parent element, not compatable with the internet explorer browser.

letter-spacing

Letter-spacing adjust the kerning of the paragraph information you input.

line-height

Line-height alters the space between new strings of text in a paragraph.

text-transform

With the text-transform property you can use uppercase, lowercase, or capitilzation to change the entire case composure of the paragraph.

word-spacing

Instead of changing the spacing of the kerning like letter-space would word spacing only changes the distance between one word and the next.

text-align

Text-align is a tag that acts like the paragraph justify would in a standard word document program. You can justfy the text to be on the left, the center, and the right.

text-indent

With this tags you can specify how much spacing you went to indent text from the begining perimeters of it's paragraph tag settings.


General Properties:

outline

Outline functions just like border functions. With the outline property you can set the color, a style, and the width. The styles are the same as border is: none, dotted, dashed, solid, double, groove, ridge, inset, outset, and inherit. Outline creates a line right outside of the border element on content.

display

Display handles the visibility of content and how it functions as an element in CSS. Meaning with display you can set an item to block or inline or you can set the display to be visible or not.

background-color

Background-color sets the whole background of the content you are applying the property to just like color but it instead effects the entire elements background.

background-image

This tag will set the background to an image by loading the image of choice to fill the content settings.

background-repeat

This will repeat the image that you have set so that you can create patterns with an image.

background-attachment

Background-attatchment will set the background in certain ways. It alters the style of what you set and places it according to the property you set. These properties include: scroll, fixed, local, initial, and inherit.

background-position

In order to set the exact position of your image you are required to use the background-position element. You can set the position in terms of justify quadrants, x%/y%, xpos/ypos, initial, and inherit.

background

To set the aforementioned properties with one setting you can use this tag which is the shorthand version of typing each individual property out.
