Lessons! Hooray for lessons.

To the right is the lessons bar or if you want to skip around you can click one of the links below.



  • Lesson 1: The Basics: What is HTML?


    • Objective: Teach basic HTML concepts to beginning learners. In this lesson we're going to discover the coding language HTML! Trust me, it's fun.



  • Lesson 2: Block Elements & Inline Elements


    • Objective: Now we know the basics of HTML. Let's explore further. The point of lesson 1 is to explain what a block/inline element is and how to use various block/inline element tags. Examples of each will be provided for user to use as reference. Learn on budding web developers.



  • Lesson 3: Advanced Block Elements & Inline Elements


    • Objective: Let's get a little bit more techinical. Next I will explain advanced block/inline element is and how to use various advanced block/inline element tags. Gear up for knowledge my beautiful webbies!.



  • Lesson 4: Forms


    • Objective: Block and inline elements mastered, check. Now let me explain what forms are and describe the various properties a developer would need. You've seen forms all over the web. Put password here, what's your gender, type a comment here: all forms of forms. Pump yourself up for more conquesting as this learning adventure continues!



  • Lesson 5: CSS Basics


    • Objective: Lesson 5 will teach you how to use CSS (a Cascading Style Sheet) to add style to your webpage. By the end of this lesson you should be familar with CSS and how it functions as a means to style a webpage. You will also become more familar with the basic CSS properties.



  • Lesson 6: Box Modeling Properties


    • Objective: Alright, so now we can style a webpage with some basic CSS tags but lets get more in depth. Have you ever heard of the box model? In this lesson we will explore the box model and it's characterstics.



  • Lesson 7: CSS Properties


    • Objective: The basics are done, the box model has been learned, so our next step is now, ta da da da... CSS Properties. We won't learn the complete list of all of the CSS properties available but we will learn the most used properties.



  • Lesson 8: CSS Positioning


    • Objective: Lesson 8 is about CSS position and how to manipulate to do exactly what you want. Positioning is extremely important for layout without it you will not be able to create the website of your dreams.
