Lesson 2 - Block Elements & Inline Elements

Objective: Now we know the basics of HTML. Let's explore further. The point of lesson 1 is to explain what a block/inline element is and how to use various block/inline element tags. Examples of each will be provided for user to use as reference. Learn on budding web developers.

What are block elements?

Block elements span the entire area of its space. A block-display element breaks the flow of design and the space it accompanies is reserved primarily for its attribute.

div

div is short for division so in essence a div is section or block of HTML used to structure pages, add properties to elements, and group blocks of content.

Lets use a div to alter the color of our text. If we type into our html editing softare this string of code:

<div style="color:#1039D7">>
	<h5>This h5 text has changed style! Thanks example!</h5>
</div>

That, and only that <h5> text will change color. (note: it changed font too. That's because of my style.css, more on that later!)

an <ol> tag is an ordered list that uses numbers to label the list.

  1. 1. Bread
  2. 2. Milk

  4. 4. Cereal

ul

an <ul> tag is an unordered list that uses bullets to label the list.

  1. • Bread
  2. • Milk
  3. • Peanut Butter
  4. • Cereal

p

a paragraph tag is used for blocks of text. <---- I've been using <p> tags this whole time.

li

a <li> tag is a list tag, list tags are used with either <ol> or <ul> tags to define the list categories

To create the ordered list and unordered list from above I used a li tag for the bread, milk, peanut butter, and cereal part.

hr

a hr tag is a tag used to separate content by use of a line.

What are inline elements?

Inline elements are elements that do not break the flow of your websites content. They happen inline therefor you can design around their function.
