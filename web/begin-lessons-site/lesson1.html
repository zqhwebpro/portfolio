Lesson 1 - The Basics: What is HTML?

Objective: Teach basic HTML concepts to beginning learners. In this lesson we're going to discover the coding language HTML! Trust me, it's fun.

What is HTML?

HTML stands for Hyper Text Markup Language. Hyper Text Markup Language is essentially the building block language used to code websites to be displayed for the World Wide Web.

What is the role of the browser?

A browser is a software application created for receiving, displaying, and communicating data across the World Wide Web. There are different types of browsers as you might be aware. From mobile, to PCs, to Macs: a browser is used for accessing the internet world.

What are the requirements of HTML tags?

1. Tags must be opened and closed:

Open the html tag: <html> ...and your html content goes here.

Then... you MUST close the html tag like this: </html>

2. HTML is written in all lowercase

Correct: <bold>Hi World!</bold>

Incorrect: <Bold>Hi World!</Bold>

Web Developers only ever using lowercase so you should to. It's quicker and it's the standard, there is no other.

3. Proper nesting is required of all tags.

Correctly done tag b's content fit nicely inside of tag a's with no error:

    <tag_a>

        <tag_b>

        </tag_b>

    </tag_a>

Within <tag_a> is nested <tag_b> and everything will function.

Incorrectly done, errors will happen:

    <tag_a>

        <tag_b>

    </tag_a>

        </tag_b>

<tag_b> is NOT nested properlly within <tag_a>! An error will occur.

4. Call out attributes in double quotes

<img src=”attribute” alt=”attribute” width=”attribute” height=”attribute” />

In the <img> tag the alt attribute that you apply would be a rollover explaination of the image, if it's a basket full of kittens then within alt put "Basket Full of Kittens!". Width and height can, but do not have to, be attributed. If an image is by default 1900 x 1280 and you need it to be smaller you can change the properties using width and height. What are the requirements of an HTML page?

All HTML documents must always have these tags: HTML tag, head tag, body tag, title tag.

Now, let's code our first website together. Open up a notepad document, save it as index.html and type the following:

<html>
<head>
<title>Document Title</title>
</head>

<body>
<h1>Hello World! (headline 1 tag)</h1>
<p>How's it goin'? (paragraph tag)</p>
</body>
</html>

Doctypes (strict vs. transitional)

A doctype defines the documentation type. It’s declared before the <html> tag however, it is not an HTML tag itself. A common Doctype tag would look like this:

<!DOCTYPE html>

A strict tag contains all HTML elements and attributes but does not include presentational or deprecated elements (like fonts). Framesets are not allowed.

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">

A strict tag contains all HTML elements and attributes and does include presentational or deprecated elements (like fonts). Framesets are not allowed.

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">

Meta tags (character sets)

Meta tags are used to provide metadata to the document. Metadata is not displayed on the website but instead is used for keywords, page descriptions, and things of that nature.
