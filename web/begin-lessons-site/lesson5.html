Lesson 5 - CSS Basics

Objective: Lesson 5 will teach you how to use CSS (a Cascading Style Sheet) to add style to your webpage. By the end of this lesson you should be familar with CSS and how it functions as a means to style a webpage. You will also become more familar with the basic CSS properties.

Applying CSS:

Inline:

Inline CSS means that you would be applying a style property and value to a tag directly within the HTML.


Example of an inline div of color:blue;.

I did that like this:

<div style="color:blue;">Example of an inline div of color:blue;.</div>

Internal:

Internal CSS implies that you are adding a style property in the head tag like this:


<head>
     <style type="text/css">
     selector {
           property: value;
     }
     </style>
</head>

External:

External CSS is created by creating a seperate CSS document and linking to it in the header. We would do that by placing the following code between our <head>.


<link href="lib/css/styles.css" rel="stylesheet" type="text/css">

CSS Structure

There are three parts to the CSS structure: The selector, properties, & values.


#selector{
		property:value;
		property:value;
}

The selector is a label for the code we are applying CSS to. The property would be what we would like to do to that code. And the value would be how much and what we are doing with the property. Let's try another example:


#examplefont{
		color:blue;
		font-family:Verdana, Geneva, sans-serif;
}

id vs class

An id is a class selector used in CSS to identify a single element by using ( # ) the hashtag symbol. A class is a class selected used in CSS to identify multiple elements by using ( . ) a period.


Precedence

Sometimes CSS properties will overlap and in this case, the most specific rule will be applied. For example, referencing a class will take precedence over referencing a general tag. Referencing an id will take precedence over a class reference.
