Lesson 3 - Advanced Block Elements & Inline Elements

Objective: Let's get a little bit more techinical. Next I will explain advanced block/inline element is and how to use various advanced block/inline element tags. Gear up for knowledge my beautiful webbies!

Block Elements

blockquote

a block element that indents a paragraph of text, used primarily for quotations.

I am a blockquote. This is an example.

dl

used to define a definition list.

For the next three elements we will reference this example:

Car
Travels by land goes vroom vroom.
Boat
Travels by sea goes fbruuuhn.
<dl>  <--- begins the definition list
  <dt>Car</dt>
 	 <dd>Travels by land goes vroom vroom.</dd>
  <dt>Boat</dt>
 	 <dd class="margbot">Travels by sea goes fbruuuhn.</dd>
</dl>  <--- ends the definition list


dt

define an item in a definition list.

<dl>
  <dt>Car</dt>  <--- item one
 	 <dd>Travels by land goes vroom vroom.</dd>
  <dt>Boat</dt>  <--- item two
 	 <dd class="margbot">Travels by sea goes fbruuuhn.</dd>
</dl>


dd

used to explain the item of a definition list.

<dl>  <--- begins the definition list
  <dt>Car</dt>
 	 <dd>Travels by land goes vroom vroom.</dd>  <--- definition of the term
  <dt>Boat</dt>
 	 <dd class="margbot">Travels by sea goes fbruuuhn.%</dd>  <--- definition of the term
</dl>


pre

everything inside the pre tag appears exactly the way it is written, indents and all. Also it is important to note that the pre tag is loaded with a default font.

Inline Elements

abbr

using the abbr tag around an abbreviation will cause a mouse over effect of what the abbreviation stands for.

But you can call me Mr. Right. (hover your mouse over Mr. to see)

acronym

using the acronym tag around an acronym will cause a mouse over effect of what the acronym stands for.

It was so funny I had to LOL. (hover your mouse over LOL to see)

cite

a citation tag that makes text that you tag italic.

Bernini's Ecstasy of Saint Teresa is a timeless masterpiece.

code

a code tag makes that text appear like techy computer text.

0101001 001010 01010 10 1001000101001


q

used to put quotations around text.

After a long dramatic pause Elizabeth spoke, The world doesn't belong to us anymore.

sub

creates subscript text little text below the base line of the rest of text.

sup

creates superscript text little text above the top line of the rest of text.
