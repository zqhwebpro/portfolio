Lesson 301 - Rollover

Need an image to change on a mouse over hover? Think about, you can display a dingy house in a before picture and then call attention to the user to hover over that said image. Then BLAMO! A beautiful after shot of that same house after an awesome remodel. It would be a great technique to involve people with your website but how...? Rollover is a jQuery function where we can do that very tactic. Let's get to coding!

Example:

Roll over this image to see what the girl from the ring looks like without her makeup.




HTML Markup:

<!-- HTML consist of a span with ID and 2 images for our jQuery fade -->

<span id="fader">
	<img id="ring" src="lib/img/ringbefore.jpg" alt="Ring Girl" />
	<class ="to" src="lib/img/ringafter.jpg" alt="Girl from Ring" />
</span>

CSS Markup:

/* Our span class needs some properties defined for our jQuery markup to work */

#fader {
  position: relative;
}

#fader .to {
  display: none;
  position: absolute;
  left: 0;
}

jQuery Markup:

$(document).ready(function() { // BEGIN 'ZE FUNCTION!

  $('#fader').hover(function() { // setting hover effect to an id

    $(this).find('img:eq(1)').stop(true,true).fadeIn(); // when first image is moused over fadeIn next.
  }, function() { // begin

    $(this).find('img:eq(1)').fadeOut(); // image fadeOut on hover
  }) // end hover

}); // END 'ZE FUNCTION!
