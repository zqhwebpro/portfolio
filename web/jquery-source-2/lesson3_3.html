Lesson 303 - Innerfade

Lesson 303 will demonstrate the innerfade function. Innerfade is a great jQuery technique you can use when you have a ton of constantly updating content. Think of it as a news ticker or an animated list of links that can connect you to content from all over the web.

Example:

Spirit Cellar News:

  • Bring your Pets to Church
  • Fundraising for the Missionaries
  • Friday Faith Day
  • Sunday Service - Calling on God


HTML Markup:

<!-- In HTML a news id has to be set up for the innerfade function -->

<div id="news">
	<h4>Spirit Cellar News</h4><
		<ul>
        	<li><a href="#">Bring your Pets to Church</a></li>
        	<li><a href="#">Fundraising for the Missionaries</a></li>					
        	<li><a href="#">Friday Faith Day</a></li>
        	<li><a href="#">Sunday Service - Calling on God</a></li>
        </ul>
</div>

CSS Markup:

/* Our ID must be styled */

#news {
  display: block;
  height: 48px;
}

#news h4{
	font-size:20px;
	font-family:'Yanone Kaffeesatz', sans-serif;
	font-weight:400;
	color:black;
	margin-bottom:10px;
}

#news ul {
  list-style-type: none;
}

jQuery Markup:

$(document).ready(function() { // document is ready

  $('#news ul').innerfade({ // the ul of the news id will innerfade
    animationtype: 'slide', // animation is set to slide
    speed: 750, // animation time is set
    timeout: 3000, // animation timeout is set
    type: 'random' // a random property random picks through ul lists
  }); // function over
  
}); // document completed, full function over
