Lesson 103 - Alert!

In this lesson, we will discover the alert function. By initiating an alert function to find and use the first paragraph as the pop-up declaration.

HTML Markup:

<!--First you must declare the text you wish to display in the pop-up alert with use of a 
paragraph function for this demonstration-->

<p>In this lesson, we will discover the alert function. By initiating an alert function to 
find and use the first paragraph as the pop-up decleration.</p>

CSS Markup:

/* CSS is not required for this paticular lesson. */

jQuery Markup:

$(document).ready(function(){ // document is ready to function

	alert($('p:first').text()); // alert is activated to call the text using 
the first paragraph tag in the HTML

}); //end of function
