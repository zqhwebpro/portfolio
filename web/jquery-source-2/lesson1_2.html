Lesson 102 - Zebra Striping

Zebra striping allows you to add style to your HTML table roles. In this demonstration I will show you how to combine HTML, with CSS, and jQuery to create a Zebra Stripped table!

Example:

Music Artist Favorite Song From The Album
Slipknot XIX .5: The Gray Chapter
Milky Chance Stolen Dance Sadnecessary
Hozier Take Me To Church Take Me To Church EP

HTML Markup:

<!--In HTML we will make our table!-->

<table class="mustbclass" id="mustbeid">
	<thead>
    	<tr>
        	<th>Music Artist</th>
            <th>Favorite Song</th>
            <th>From The Album</th>
        </tr>
    </thead>
	<tbody>
		<tr>
    		<td>Slipknot</td>
        	<td>XIX</td>
        	<td>.5: The Gray Chapter</td>
    	</tr>
   		<tr>
    		<td>Milky Chance</td>
        	<td>Stolen Dance</td>
        	<td>Sadnecessary</td>
    	</tr>
    	<tr>
    		<td>Hozier</td>
        	<td>Take Me To Church</td>
        	<td>Take Me To Church EP</td>
    	</tr>
	</tbody>
</table>

CSS Markup:

/* In CSS we will style our tables so our content fits better within our specifications */

table {
	border-collapse:collapse;
	font-size:12px;
	margin:0 20px 20px 20px;
	border-top:2px solid #015287;
	width:480px;
}

th {
	border-bottom: 2px solid #015287;
	color: #D15600;
	font-size: 14px;
	font-weight: normal;
	text-align: left;
	padding: 3px 8px;
}

td {
	padding: 6px;
}

$(document).ready(function(){ //this function is to ready // document is ready to function
