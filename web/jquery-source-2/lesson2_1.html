Lesson 201 - Color Animation

Color animation is a useful jQuery script that can be used to draw attention to special text in a fun and unique way!

Example:

Contents on this websites are copyrighted by website creators 2014.


HTML Markup:

<!--The first step is to input the text that will highlight with a div id-->

<p id="ptext202>Contents on this websites are copyrighted by website creators 2014.</p>

CSS Markup:

/* a little bit of padding was used to aid the design */

#ptext201 {
	padding-left:20px;
}

jQuery Markup:

$(document).ready(function(){ //ready the function

	$('#ptext201').animate({'backgroundColor':'#ff9f5f'}, 2000); //on id ptext201 animate 
a background color on timer of 2000

}); //end the function
