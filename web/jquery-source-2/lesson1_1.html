Lesson 101 - Slide Toggle

For the first lesson in the series we will be learning how to code a simple slide toggle. A slide toggle can help add graphic pizzazz and functionality to your site by hiding and showing snippets of information.

Example:

Rain cancelled due to the sun.


HTML Markup:

<!--First you must declare the text you wish to hide and show using a paragraph tag in HTML-->

<p id="slidetext">Press my buttons!</p>

CSS Markup:

/* Assign a class of .slidedemo to these properties to create the teal rounded-corner box */

.slidedemo{
	padding:15px;
	background-color:#9AF1E4;
	border-radius: 10px;
	-moz-border-radius: 10px;
	-webkit-border-radius: 10px;
}

/* Assign an ID of #slidetext to these properties to make the text the font you want it and the color */

#slidetext{
	font-family: 'Oxygen', sans-serif;
	color:black;
}

jQuery Markup:

$(document).ready(function(){ // document is ready to function

$('#hide-button').click(function(){	// add click function to div #hide-button
	$('#slidetext').slideUp();	// slideUp to hide all div's with id #text
}); // end hide-button function
	
$('#show-button').click(function(){	// add click function to div #show-button
	$('#slidetext').slideDown(); // slideDown to hide all div's with id #text
}); // end show-button function
	
}); // end function
