Lesson 302 - Cycle Fade

I know you've seen this cycling images before. Occasional, websites will display image content below headers that go from one picture to the next. It's a great way to use changing images to maintain user interest! Cycle is just the trick to do that very thing. Cycle is capable of doing multiple transitions including fade, shuffle, zoom, curtainX, and many more! The various styles of transition effects can be located here: http://jquery.malsup.com/cycle/.

Example:

Come to see the sights and wonders of Egypt!




HTML Markup:

<!-- Create a div of photose and within it place our cycling images -->

<div id="photose">
	<img alt="Pyramid" src="lib/img/egypt1.jpg" />
	<img alt="Camel" src="lib/img/egypt2.jpg" />
	<img alt="Sphynx" src="lib/img/egypt3.jpg" />
	<img alt="Temple" src="lib/img/egypt4.jpg" />
	<img alt="Dark Sea Dive" src="lib/img/egypt5.jpg" />
</div>

CSS Markup:

/* Set a size constraint in CSS for layout purposes */

#photose:
	height: 400px;
}

jQuery Markup:

$(document).ready(function(){ // document is read

  $('#photose').cycle({ // adding cycle to the photos div id
    fx: 'shuffle' // setting the effect fade 
  }); // end cycle

}); // end document
