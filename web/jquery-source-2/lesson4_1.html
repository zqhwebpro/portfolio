Lesson 401 - Expandable Menu

By using drop down arrow graphics on links we can create a stylized expandable menu to show case our navigation menu.

Example:

  • Need help?
    • Look here
    • Or here
  • Other Links
    • Link 1
    • Link 2
  • About
    • The who
    • The what
    • The why






HTML Markup:

<!-- In HTML we have to set a navigation list -->
     
<nav role="navigation">
	<ul id="menu401">
		<li><a href="#">Need help?</a>
			<ul class="active">
	      		<li><a href="#">Look here</a></li>
		     	<li><a href="#">Or here</a></li>
		  	</ul>
	      </li>
	      <li><a href="#">Other Links</a>
		  	<ul>
		    	<li><a href="#">Link 1</a></li>
		      	<li><a href="#">Link 2</a></li>
		  	</ul>
	      </li>
	      	<li><a href="#">About</a>
		  		<ul>
		      		<li><a href="#">The who</a></li>
		      		<li><a href="#">The what</a></li>
		      		<li><a href="#">The why</a></li>
		  		</ul>
	      </li>
	  </ul>
</nav>

CSS Markup:

/* Our ID menu must be styled in CSS, Don't forget to add your own arrow.png image*/

#menu401 { 
	float: left;
  	width:200px;
  	position:absolute;
  	background-color:white;
}

#menu401, #menu401 ul{
  	list-style-type: none;
  	padding: 0;
  	margin: 0;
}

#menu401 li { 
  	cursor:pointer;
  	background-image:url(../img/arrows.png);
  	border-bottom:1px solid #444;
  	background-repeat: no-repeat;
}

#menu401 li a {
  	text-decoration: none;
}

#menu401>li>a { 
    padding: 2px 35px;
	font-weight: bold;
}

#menu401 li li{ 
  	cursor:auto;
  	border:0;
  	padding:0 14px;
  	background-color:#fff;
  	background-image: none;
}

jQuery Markup:

$(document).ready(function(){ // start the function
$( '#menu401 > li > ul' ) // on menu li ul properties to function
	.hide() // begin with inner information hidden
	.click(function( e ){ // on click
		e.stopPropagation(); // open menu
	}); // end function on menu li ul property
	
  $('#menu401 > li').toggle(function(){  // begin menu li toggle function
	  $(this) // on this happening
      .css('background-position', 'right -20px') // css bg-pos will look like this
      .find('ul').slideDown(); // find ul so that we can slide it down
  }, function(){ // close primary function, begin secondary function
  	$( this ) // on this happening
      .css('background-position', 'right top') // css bg-pos will now look as so
      .find('ul').slideUp(); // find ul so that we can slide it back up
  }); // end function on menu li
}); //end full function
