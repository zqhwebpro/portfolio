Lesson 203 - Splitter

Splitter is a highly organized method of detailing information in a limited amount of space. The function neatly seperates two strings of data and places a scroll bar within it's contents to keep large paragraphs of text highly condensed.

Example:

  1. Splitter in Depth
  2. HTML
  3. CSS
  4. jQuery

jQuery Markup:

$(document).ready(function(){ //begin function
	$('#splitter > div:first').resizable({ //first splitter div is resizable and called upon
		handles: 'e',  //icon property of resize call out
		minWidth : '100', //min width of sizable content
		maxWidth : '400', //max width of sizable content
		resize: function() { //resize function begins
        
		var remainingSpace = $(this).parent().width() - $(this).outerWidth(); //the following are the dimentions 
		and properties of the resize function of the splitter
		var divTwo = $(this).next();
		var divTwoWidth = remainingSpace - (divTwo.outerWidth() - divTwo.width());
		divTwo.css('width', divTwoWidth + 'px');
    } //end resize function
  }); //end splitter function
}); //end ready function
