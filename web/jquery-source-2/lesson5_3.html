Lesson 503 - Calendar Date Picker

By combining some extra CSS and mixing it with jQuery after learning this lesson you will be able to create a pop-up calander on to your website. Users can then see the date and pick it accordingly.

Example:

Pick a date for your appointment




HTML Markup:

<!-- In HTML we will use some text and an input type to start -->
    
<p id="date-pick-ex" class="editable">Pick a date for your appointment</p>
      <input type="text" id="date" />

CSS Markup:

/* We will be using CSS to add some margin to the top for layout purposes */

.ui-datepicker-trigger {
	margin-top: 3px
    ;
}

/* For this lesson you will also need to link jQuery UI CSS Framework 1.8.17
from the website http://docs.jquery.com/UI/Theming/API */

jQuery Markup:

$(document).ready(function(){ // document is ready

	$('#date').datepicker({ // adding datepicker to the element #date or select #date and add datepicker to it
		showOn: 'both', // show on both
		buttonText: 'Choose a date', // button text
		buttonImage: 'lib/img/calendar.png', //add calendar image to button
		buttonImageOnly: true, //button is image only
		numberOfMonths: 2, //two months set
		maxDate: '0d', //max date 0 days
		minDate: '-1m', //min date -1 days
		showButtonPanel: true //shows true
	}); //end function datepicker

}); //end entire function
