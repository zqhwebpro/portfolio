The Chevrolet.

 

$(Rollover) Chevrolet

 

The Scenario:

A clients wants to show off their newest used car but they have limited space. There are two photos they would like to showcase but only want it to fill one space. So what can be done?

Rollover will allow the user to mouse over the images and they will be able to watch them change on a timer, it's a great solution for the restrictions.

HTML:

HTML is used in this demonstration to load the images and run the scripts you will need this library linked to, if you forget how to do that you can go back to Doc Ready under the training link to review. For this example of lightbox a span tag will be placed around our images, this will allow us to position correct in CSS. The code for that entire process looks like this:

<!--Load the image onto the website with a button-->

<span id="fader">
    <img src="lib/img/chevy1.jpg" alt="Red Chevy" width="290" height="217" id="chevy" />
    <img class="to" src="lib/img/chevy2.jpg" width="290" height="217" />
</span>

 

CSS:

With the span we have specified an id of fader and on the image there is a class of to. The reason for this is to position both elements correct while hiding the image with a class. jQuery will find that class and load it's script when initiated. It looks like this:

#fader {
  position: relative;
}

#fader .to {
  display: none;
  position: absolute;
  left: 0;
}

 

jQuery:

With the fader id and to class in place we can call a fadeIn action on a hover to load the ".to" class. By setting up a simple equation we will cause it to both fade in and fade out on a hover.

$(document).ready(function() { // doc ready
  $('#fader').hover(function() { //on class of #fader call on a .hover function
    $(this).find('img:eq(1)').stop(true,true).fadeIn(); //find and image equal to (1) more, stop with
    true, true from looping, and begin a .fadeIn action
  }, function() { //finish one function and loads another this factor
    $(this).find('img:eq(1)').fadeOut(); //find that image and fade it back out
  }) //end functions
}); //end script

 

Rollover Chevrolet

In order to show both striking shots of the clients car within a restrained amount of space a Rollover was created. The rollover will fadeIn on a hover and fadeOut on a mouse release. Now the client is happy and we have made millions of dollars.
