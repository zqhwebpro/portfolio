Haircut Appointment $(Date Picker)

The Scenario:

A hairstylist has began using the web to draw in customers to their business. They want they're customers to be able to set up their own appointments online. To do so, there is a custom calander CSS file we can use to display a date picker.

Calling upon a small input type code within HTML with an id of date and uploading the custom script will allow the user to select the button and choose a date. The CSS file is customizable so you can change the look of it.

HTML:

First you must download and call upon the CSS script, you can find that CSS file here:

jquery-ui-1.8.17.custom.min.js

 

Just save it to the appopiate file location and call upon it in the header. Aftwards, we will be adding an input type with an id within the body section.

<head>
    <link href="lib/css/jquery-ui-1.8.17.custom.css" rel="stylesheet" type="text/css">
    <script src="lib/script/datepicker.js"></script>
     <script src="../../lib/jquery-ui-1.8.17.custom.min.js" type="text/javascript" charset="utf-8"></script>
</head>

<input type="text" id="date" />

 

CSS:

Only minor CSS is necessary to adjust the the margin. The jquery-ui-1.8.17.custom.css file is customizable so feel free to change it to match your needs.

.ui-datepicker-trigger {
    margin-top: 3px;
}

 

jQuery:

The date picker script finds the date id that is set in our input code within HTML and we are able to set certain properties. The amount of months, the images, and the days where you can pick; they may be altered. The script looks like so:

$(document).ready(function(){ // doc ready
    $('#date').datepicker({ // find the class of date
        showOn: 'both', // show both methods is on
        buttonText: 'Choose a date', // the button text should say Choose a date
        buttonImage: 'calendar.png', // load the calendar.png
        buttonImageOnly: true, // button image is set to image only
        numberOfMonths: 2, // show 2 months
        maxDate: '0d', // max date of 0 days
        minDate: '-1m', // min date of -1m
        showButtonPanel: true // the button panel show is true
    }); // close up datepicker action
}); // close up doc

 

Date Picker:

An input method, and a jquery script were able to transform a custom CSS file into an online calander where users can select specific dates.
