Class President $(Login Panel)

 

The Scenario:

A client is teaching their first-graders how to cast internet votes with a class president election. They have come to you with the task of creating a unique and interesting way to submit they're choice.

By implementing a form we can create user driven content ready to send through the internet through the click of a submit button. With jQuery we can script that interesting and fun vote cast the client is after.

HTML:

In HTML we will be creating the id's and forms required to set up the vote box. This specific lesson also requires an additional library to add to the header ALONG with the 1.7.1.min.js library we've been calling upon you can find that link here:

Direct jquery-1.8.17.min.js Link

Call upon those librarys and the login panel script we'll be coding in the head and then add the body content like so:

<!--Load the scripts into the head-->

<head>
    <script src="lib/script/jquery-1.7.1.min.js"></script>
    <script src="lib/script/jquery-1.8.17.min.js"></script>
    <script src="lib/script/login.js"></script>
</head>


<!--Load the image onto the website with a button-->

<div id="login">
    <a href="#">Click to submit a vote!</a>
        <form action="">

jQuery:

This login panel script will run standard actions we've used before like: animate, duration, easing, & other basic functions to make our selectors respond with fluidity. In this lesson you will be introduced to the .prev method with will find all of the preceding siblings of the selector. Here is the script:

$(document).ready(function(){ // doc ready
$('#login form').hide(); // hide the login form
  $('#login a').toggle(function() { // login a has a toggle of function
    $(this) // this will run
      .addClass('active') // add a class of active to the element
      .next('form') // find next form element
      .animate({'height':'show'}, { // animate the height to show the content
        duration: 'slow', // animate it slowly
        easing: 'easeOutBounce' // animate it with an easeOutBounce
      }); // stop this function
  }, function() { // next function
    $(this) // this function if not other
      .removeClass('active') // remove the active class
      .next('form') // on next form element
      .slideUp(); // animate it with a slideUp
  }); // stop this, this
  $('#login form :submit').click(function() { // activate the submit element of the
  form with a click function
    $(this) // then this
      .parent() // the parent element
      .prev('a') // find immediate preceding sibling of each element with 'a'
      .click(); // click event listener
  }); // end #login form :submit
}); // end script
