Top Reggae Hits $(Load with Selector)

 

The Scenario:

A client is a reggae radio station from in the area and they have come to you asking for a website that will display the top 5 Reggae hits of the week.

With load with selector we can create condendensed HTML files that will load through jQuery to display the information styled below links to load them.

HTML:

Same head again, remember to write in your new jquery script to the html file. Otherwise, we will be using HTML to set up the content shown in the above example.

<head>
    <script src="lib/script/jquery-1.7.1.min.js" type="text/javascript" ></script>
    <script src="lib/script/lwselector.js" type="text/javascript"></script>
</head>

<p>Top 5 Reggae Hits of the Week 5/14/2015:</p>
        <div id="i_service">
                <ul>
                    <li><a href="choice_e.html">Click to see Pick Five</a></li>
                    <li><a href="choice_d.html">Click to see Pick Four</a></li>
                    <li><a href="choice_c.html">Click to see Pick Three</a></li>
                    <li><a href="choice_b.html">Click to see the Runner Up</a></li>
                    <li><a href="choice_a.html">Click to see the #1 Pick</a></li>
                </ul>
                <div id="service">
                    Click an above selection for more information.
                </div>
        </div>

 

CSS:

For this examplem style the fonts and links to customize the look. The example above was created with the following code:

#i_service a{
    font-weight: 700;
    font-family: 'Open Sans', sans-serif;
    line-height:1.5;
}

#service{
    margin-top:15px;
    font-family: 'Open Sans', sans-serif;
    font-size:12px;
}

#ans_service{
        font-family: 'Open Sans', sans-serif;
    font-size:12px;
}

 

jQuery:

When we load with a selector we are using jQuery to script code that will, basically, generate a url where the script is loading. To do that, we have to use click functions, generate events, call on variables and set up an html loading string of code. Below is a commented out version of that basic concept:

$(document).ready(function(){ // doc ready
  $('#i_service ul a').click(function(e) { // load class i_service ul a and add click function generate an
  (e) event
    var url = $(this).attr('href') + ' #ans_service'; // load variable url this then runs
    .attr, get value of href, and + class ans_service
    $('#service').html('loading...').load(url); // selector open class service, in html
    script - state test loading in place holder spot, then immedeiately load the url and
    html content in it's place
    e.preventDefault(); // the event may not ever Default
  }); // close up click function actions
}); // doc close
