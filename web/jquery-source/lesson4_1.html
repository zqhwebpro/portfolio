Trivia Box $(Accordion)

The Scenario:

A client has come to you with the task of creating a script to run a trivia game on their website! They would like the trivia question and answer to take up the same amount of space on the page but the answer portion must load hidden!

For this we will be implementing a simple accordion script to call upon HTML and CSS elements to hide the trivia answers.

HTML:

The same 1.7.1.min script that we have been using must be called upon in the head of the HTML. In the body we will be coding the question, answer, and id's required of us to style the trivia box. A div with an id of "accord" has been called out: we will need this div to run our jQuery command.

<div id="accord">
    <ul id="accordion">
        <li class="active">Question #1:
            <ul>
                <li><p>Ralph Waldo Emerson was an American writer who
                was famous for leading
WHICH movement of the mid-19th century?</p></li>
             </ul>
        </li>
        <li>Answer:
            <ul>
                <li><p>The Transcendentalist movement.</p></li>
            </ul>
        </li>
     </ul>
</div>

 

 

CSS:

The styles we will need to set up for the Trivia Box Accordion are color properties, paddings, and special interactive functions (like a cursor change). An active class is set up to change the element it's specified to. I have used standard paragraph tags for the question and answer. It looks a little something like this:

#accordion {
    width:910px;
}

#accordion li{
    cursor: pointer;
    font-weight:700;
    color:#015287;
    border:1px solid #b2b2b2;
    margin-bottom:5px;
    list-style-image:none;
    list-style-position:outside;
    list-style-type:none;
    -moz-border-radius: 4px;
    -webkit-border-radius: 4px;
    padding: 5px 10px;
    font-family: "Open Sans",sans-serif;
}

#accordion li p{
    color:black;
}

#accordion li.active{
    color:#C30;
}
#accordion li ul{
    padding:0;
    margin:10px 0 0 0;
}

#accordion li.active li{
}

#accordion li li{
    font-weight: normal;
    background:none;
    border:0;
}

 

 

jQuery:

The script that we are running is going to call on an event of stopPropagation. stopPropagation prevents any parent handlers of the element we call out from being notified of the event preventing any elements other than the one specified to be notified of the scripts method. By using click event listeners and slideToggles the script will find our element and execute it's function.

$(document).ready(function(){ // doc ready
  $('#accord ul > li ul') // id accord ul will find everying that is li ul
    .click(function(event){ // click event listener
      event.stopPropagation(); // run the stopPropogation to prevent parent
      element interference
    }) // end first click function
    .filter(':not(:first)') // filters everything that is not first in the sequence
    .hide(); // hide #accord ul > li ul after executing event listener

  $('#accord ul > li').click(function(){ // id accord ul will find everying that is li
    var selfClick = $(this).find('ul:first').is(':visible');
    // set variable of selfClick that equates to a find and visible status of the first ul in it's sequence
    if(!selfClick) { // if statement instead of this
      $(this) // this will run on if
        .parent() // get parent of element in this statement
        .find('> li ul:visible') // find the > li ul:visible
        .slideToggle(); // slide toggle the whole element
    } // end if

    $(this) // this is the other statement in the if field
      .find('ul:first') // find the first ul
      .stop(true, true) // stop it from what it is doing
      .slideToggle(); // run a slideToggle
  }); // end #accord ul > li
}); // end script

By using a relatively simple accordion script jQuery has allowed the use of a trivia box just like the client wanted! The trivia ended up being so popular that the client went on to become a host of a trivia night at a local bar that features that most difficult trivia questions in the County.
