D20 $(Check All)

 

The Scenario:

A client needs your help to create a Dungeon & Dragon's fansite. On the website the client plans to obtain a lot of user data about the game.

Using a check-all selector in jQuery multiple lines of data can be selected all at once. Most helpful for the style of website being created.

HTML:

Call out your script in the head and build a form. In the form make sure that the check all option has a class of check-all, that way jquery can be called upon to check all of the options. 

<head>
	<script src="lib/script/checkall.js" type="text/javascript"></script>
</head>

<form action="">
        <div class="d20class">
          <h3>Which D20 class have you played before?</h3>
          	<input name="class" type="checkbox" value="fig" />Fighter<br/>
          	<input name="class" type="checkbox" value="mag" />Mage<br />
          	<input name="class" type="checkbox" value="hea" />Healer<br />
          	<input name="class" type="checkbox" value="rog" />Rogue<br />
          	<hr />
          	<input class="check-all" name="class" type="checkbox" /><span>Check all</span>
         	 <br/><br/>
			<input type="submit" value="Submit" class="submit" />
        </div>
</form>

 

CSS:

No CSS code is required of this example:

 


                    
jQuery:

          
For this script an event listener is set to create variables. This variables are labeled as group and the group then changes value from unchecked to checked. It looks like this:


$(document).ready(function(){ // doc ready
  $('.check-all:checkbox').change(function() { // check-all is selected for a state of
  change to occur in a function
    var group = ':checkbox[name=' + $(this).attr('name') + ']'; // a variables group is
    created for checkbox
    $(group).attr('checked', $(this).attr('checked')); // the value for the variables of
    group now, are changed to all be checked
  }); // close check-all function
}); // close doc ready

 



D20 Check All:

                    
By coding a form in HTML jQuery was scripted to change any unchecked item to checked. We used a change function to set the value of the group variables we called upon in the script.
