UFC $(Innerfade)

 

The Scenario:

A client has a UFC fan site setup but they really wish they could have a news ticker on it, that way users could select from the most popular headlines to read from. What to do, what to do...

By coding an innerfade jQuery function for our client we can help them out with their dilemma.

HTML:

We will need to call upon the jQuery innerfade library for this you can find that here:

jquery.innerfade.js External Link

After you have downloaded and uploaded that script you can set up a div with an unorder list for our UFC news ticker.

<!--Load the new script into the head-->

<head>
        <script src="lib/script/jquery.innerfade.js"></script>
</head>


<!--Create the UFC Headline content-->
<div id="ufc">
    <h3>UFC Headlines</h3>
        <ul>
            <li><a href="#">Alptekin Ozkilic: Dreams and Visions of the Turkish Delight</a></li>
            <li><a href="#">UFC Execs Outline Athlete Outfitting Policy</a></li>
            <li><a href="#">Sean O’Connell: Making a Name for Himself</a></li>
            <li><a href="#">Pettis out of Jury Fight in July</a></li>
        </ul>
</div>

 

CSS:

In CSS we will set up our div and ul.

#ufc {
  display: block;
  height: 70px;
}

#ufc ul {
  list-style-type: none;
}

 

jQuery:

The innerfade library is loaded therefor we can call upon it's action and specify different functions for it. The ticker we are setting up is set to be a random sequence of headlines that will slide over one another.

$(document).ready(function() { // doc ready, function start
  $('#ufc ul').innerfade({ // on id of ufc ul call on .innerfade action
    animationtype: 'slide', // the animation type called on is slide
    speed: 'normal', // the speed of sliding is a normal pace
    timeout: 3000, // time between slides is set at 3000 milliseconds
    type: 'random' // the type of rotation will be random
  }); // end innerfade
}); // end document

 

UFC Innerfade:

Now we have a ticker set up for our client, they loved it and the fans are in touch with the latest UFC news. With an innerfade we can create a function that will set an animation type, timer, and sequence and carry out a series of events specified by HTML and CSS.
