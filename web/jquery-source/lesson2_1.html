Where are the Gingerbread people?

 

$(fadeOut) Gingerbread People

The Scenario:

A clients has come up to me looking for an HTML web-based game for children. By clicking the image, the image should disappear. The solution: an image that fades out on click by using the jQuery action fadeOut.

jQuery is a short-script Javascript library to specialize website functionalization by loading scripts. Web designers use it for effects, animation, as a utility, for selector specialization, and other specialty features. It is the ultimate goal to add site content through HTML, stylize that content with CSS, and then apply a jQuery script to bring the libraries functionality to use. Exploring deeper, will learn to add an effect and adjust the properties of the effect.

HTML:

HTML is used in this demonstration to load the image and run additional scripts. ***It is important that you load the -min- script. I'm using a special short-script library to call out commands with. So note the HTML and make sure you get this file loaded to your page:

Direct jquery-1.7.1.min.js Link

Now that we have a library of functions we can create our own specialized code and call upon that. The code for that entire process looks like this:

<!--Load the scripts into the head-->

<head>
    <script src="lib/script/jquery-1.7.1.min.js"></script>
    <script src="lib/script/fadeout.js"></script>
</head>


<!--Load the image onto the website with a button-->

<img id="gbmen" src="lib/img/gbimage.jpg" alt="Gingerbread People" />
<input type="button" id="gbhide" value="Click The Button To Hide" />

 

CSS:

Even though ids and classes have been called upon jQuery will recognize the names to execute this script. If we had to customize further we CSS can become implemented for this but not used like that for this specific lesson.

jQuery:

HTML has loaded our script, now, we have to input the code to make it fadeOut with the timing we customized. It looks like this.

$(document).ready(function(){ //get the document read to function
 $('#gbhide').click(function(){ //initiate #gbhide to be ready for a click function
  $('#gbmen').fadeOut('300'); //intiate #gbmen and call an effect to fadeOut with a timing
  that I've added of 300ms
 }); //close second sequence ready
}); //close first sequence ready

 

Note that everything is noted by the // marker. This is so you understand the lingo of jquery. Each lesson is designed to keep you in the know about industry pratices and how to stay on top of your game.
