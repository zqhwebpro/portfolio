Personality Test $(Client-side Templating)

The Scenario:

A client who runs a Occupation & Tranquility Center has come to you with a task. She has a test ready to use on her Center's website but needs a way of processing the data with a client-side response. A client-side response means that, on a users interaction with a part of a website, that a client scripted response will output with client generated data.

To acheive our clients task we will have to use Client Side Templating as the conept that will complete the idea. (Note: client-side scripting is in contrast to server-side scripting. server-side scripting is server generate data)

HTML:

In the head don't forget the library and script, it is referenced below. For this example, you will need to code a table that would generate our clients requested data. All referenced below.

<head>
    <script src="lib/script/jquery-1.7.1.min.js" type="text/javascript" ></script>
    <script src="lib/script/cstemplating.js" type="text/javascript"></script>
</head>

<div id="cstemp">
        <p>Your personality test results:</p>
          <table width="800" height="77" id="cstable">
              <thead>
                <tr>
                  <th>Your Type:</th>
                  <th>Future Occupation:</th>
                  <th>Psychic Vibe:</th>
                </tr>
              </thead>
              <tr class="template" style="display:none;">
                <td><span class="cs_type">type</span></td>
                <td><span class="cs_occ">occ</span></td>
                <td><span class="cs_vibe">vibe</span></td>
              </tr>
            </table>
</div>

jQuery:

For this script create a function of template to load the strings of data that will be generated. Then it loads the script that will run to display the strings of data for the user to view.

function template(row, cstable) { // load a function for templates, rows, and the class
  row.find('.cs_type').text(cstable.type); //find row cs_type with the text id of type
  row.find('.cs_occ').text(cstable.occ); //find row cs_occ with the text id of occ
  row.find('.cs_vibe').text(cstable.vibe); //find row cs_vibe with the text id of vibe
  return row; // returns a value of row
} // closes up function template

$(document).ready(function() { // loads doc ready function
  var newRow = $('#cstable .template').clone().removeClass('template'); // varilable
  of each new row creates a new jquery assigning a new selector of class cs table
  id template. clone copies the method, remove class of template finishes the string
  var cstableItem = { // variables of cstableItem called upon
    type: 'EFJT', // the type selector data
    occ: 'Web Design', // the occ selector data
    vibe: 'Relaxed' // the vibe selector data
  }; // close variable
  template(newRow, cstableItem) // template of newRow and csItem
    .appendTo('#cstable') // appends, produces to class of cstable
    .fadeIn();  // generate data with a fadeIn
}); // close variable and entire function
