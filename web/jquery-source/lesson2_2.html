On Sale $(Color Animation)

 

The Scenario:

A client needs you to figure out how to highlight a sales tag in a cool and interesting way! The client says that it has to a red color and it must be animated. The solution: Using a background color animate trigger we will be able to acheive the effect the client desires.

Color animation will require a certain library for the website to load so keep that in mind. This will run an animate function of background color and executed the code for on load.

HTML:

In HTML we will define the sale content that has a class. Also, we will have to tell our website to load a new jQuery UI which you can locate here:

jquery-ui-1.8.17.custom.min.js External Link

You must save that to your scripts library or call upon it externally. Also, I am running a google fonts script because of the CSS that is set up so we have to make sure to do that too.

<!--Load the new script into the head-->

<head>
        <link href='http://fonts.googleapis.com/css?family=Bree+Serif' rel='stylesheet' type='text/css'>
        <script src="lib/script/jquery-ui-1.8.17.custom.min.js"></script>
        <script src="lib/script/jquery-1.7.1.min.js"></script>
    <script src="lib/script/coloranimation.js"></script>
</head>


<!--Load the text with an id on it-->

<p id="demo2">ON SALE: $10.99</p>

 

CSS:

The class that is set up has custimization on it like our Google Font, padding, and a size: for emphasis.

#demo2 {
  font-family: 'Bree Serif', serif;
   padding:20px;
   font-size:17px;
}

 

jQuery:

HTML has loaded our script, now, we have to input the code to make it fadeOut with the timing we customized. It looks like this.

$(document).ready(function(){ //ready function
 $('#demo2').animate({'backgroundColor':'#bd7c70;'}, 10000); //the id of demo2 will
 animate a background color to a red type of hue
}); //end ready
