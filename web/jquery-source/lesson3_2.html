The Puppies!

 

Puppy $(Slideshow Fade)

 

The Scenario:

A client has a faithful dog who just had puppies and they would like a slideshow of them for a personal site. They came to you looking for help, your answer?

A puppy slideshow fade! With a slideshow we can showcase the new liter of puppies for the world to see.

HTML:

In HTML we will be setting up an id around all of our images of "pups" that way we can set up CSS properties. On your first image, specify a class of "show" this will come into play when we write the jQuery script:


<div id="pups">
    <img alt="Henry Barkington" class="show" src="lib/img/pup1.jpg" />
    <img alt="Junior, Pickles, and Snot" src="lib/img/pup2.jpg" />
    <img alt="Bloop" src="lib/img/pup3.jpg" />
    <img alt="Shwing" src="lib/img/pup4.jpg" />
</div>

 

CSS:

By setting up position properties and display properties we can show just one image and use the jQuery script to create the actions it will take to make them play the slideshow fade.

#pups img {
  display: none;
}

#pups .show {
  display: inline;
}

 

jQuery:

The action we will be loading is called slideShow. From load we will set up it's properties to run a string of variables to play the slideshow that follows these specifications:

$(document).ready(function(){ // doc ready
  slideShow(); // intiate slideshow function call out
}); // end doc ready

function slideShow() { // call back up slideShow(); function and begin
  var current = $('#pups .show'); // check variables of #pups div id and .show class
  var next = current.next().length ? current.next() : current.siblings().first(); // run variable sequence
from current to next

  current.hide().removeClass('show'); // remove .show class from one image
  next.fadeIn().addClass('show'); // fade in next image with .show class

  setTimeout(slideShow, 4000); // timer of slideshow set to 4000 milliseconds
} // end function slideShow

 

Puppy Slideshow Fade:

With HTML we could specify id's and classes so that in CSS we could position and hide elements correctly. With the jQuery we used a slideshow function to create a variable defined sideshow of puppies. The client is so happy and her puppies have over 10,000 likes.
