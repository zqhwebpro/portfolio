Dating Site $(Sliding Overlay)

 

The Scenario:

A client is creating a dating website but they want it to do be quick and easy to use. The client wishes there was a way where you can mouseover a picture and be prompted to talk to special picks. Sliding overlay is a great way to acheive this effect.

With a sliding overlay we can popup content anywhere on this page without it being annoying, you know? Like those annoying popups that get in the way of our clicks. We can position and style it the way we want so that it does not interfere with our users browsing experience and it looks nice too.

HTML:

Just the 1.7.1.min.js library is needed for this one but we will need to add a div and some content within that div for the script to interact with. What I have done looks like this:

<div id="datingcontent">
    <div class="dating">
        <p>Mouseover to find out if she's into you.</p>
        <a><img src="lib/img/b.jpg" alt="just me :) b n goofy"
        width="256" height="329" /></a>
    </div>
</div>

 

CSS:

In CSS we will set up positions and style our overlay. The overlay will be styled for our purposes to drop down from the top of the page. That code looks like this:

#datingcontent {
  position: relative;
}

.dating a {
}

.dating a:hover, .dating-hover {
  width:105%;
  height:105%;
}

#overlay {
    position: absolute;
    width: 100%;
    height: 100px;
    top: 0;
    left: 0;
    color: #fff;
    background-color: #000;
    display: none;
    text-align: center;
}

#overlay h1 {
    font-weight: 700;
    font-family: 'Open Sans', sans-serif;
    color:white;
    font-size:28px;
    margin-top:30px;
}

#overlay p {
    margin-top:22px;
}

#overlay p a {
    font-weight: 700;
    font-family: 'Open Sans', sans-serif;
    color:white;
    font-size:18px;
}

 

jQuery:

The innerfade library is loaded therefor we can call upon it's action and specify different functions for it. The ticker we are setting up is set to be a random sequence of headlines that will slide over one another.

$(document).ready(function(){ // doc ready
  $('<div></div>') // call upon and create an empty div element
    .attr('id', 'overlay') // give it the attribute of id="overlay"
    .css('opacity', 0.65) // in css change the opacity property to 0.65 instead of 0
    .hover(function(){ // hover event listener function beings here
      $(this).addClass('active'); // this adds the class of active
    }, function() { // next function
      $(this).removeClass('active'); // this removes class of active
      setTimeout(function(){ // setTimeout function
        $('#overlay:not(.active)').slideUp(function(){ // when active = not it will slideUp
          $('a.dating-hover').removeClass('dating-hover'); // remove the dating-class
          on the hover
        }); // end setTimeout funtion
      }, 800); // slideUp speed of 800 milliseconds
    }).appendTo('body'); // appends the overlay to the body tag

  $('.dating a').mouseover(function(){ // on mouse over of the dating a id
    $(this).addClass('dating-hover'); // this will add class of dating-hover
    $('#overlay:not(:animated)') // function for when overlay is not animated
      .addClass('active') // add a class of active
      .html('<h1>Shes into you dude!</h1><p><a href="#">Send a wink.</a>
      <a href="#">Email her now.</a></p>') // add this html to the overlay
      .slideDown(); // the overlay slidesDown here
  }); // close dating a function

}); // end doc ready
