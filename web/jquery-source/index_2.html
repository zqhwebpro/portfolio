How To

Demo

At the top of the lessons pages will appear the demonstration that the scenario and code come from. Occasionally, I will update the demos to show you multiple different styles for the scenarios explained page by page.

Scenario

I explain out some scenarios here involving real life situations and then walk-you through the thinking process as if we were one. Essentially, I want you to know that you will have to figure yourself out of problems in this line of work so get ready to decipher codes to meet the request you're creating. Learning the code might be somewhat intimidating once you go it on your own but just know that there is a world of web developers that have had the same questions of you. Don't let anything get in the way of your creativity.

Code

HTML and CSS are explained and shown. Also, the jQuery code gives explinations but technical annotation describes what each line in the jQuery script does. Each lessons end with a recap and key points. Lessons are also located in the footer, and up above. Feel free to contact me with any questions. I'll have no problem clearing anything up for you.
